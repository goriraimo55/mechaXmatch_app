# ⚙️ MechaXMatch — 高専生の技術者ギルド

高専生がゲーム感覚で機械設計・ものづくり・CAD・実験・加工・AI/DXなどの実務スキルを学びながら、
企業から投稿された小さな技術タスクに「クエスト」として挑戦できる、学習 × 仕事マッチングWebアプリの**プロトタイプ**です。

> このアプリの本質は、**「企業の小さな技術課題」を「高専生の成長クエスト」に変換する**ことです。
>
> - 仕事をすると経験値が貯まる ⚔️
> - 学ぶとスキルが伸びる 📖
> - 実績がポートフォリオ・スキル証明書になる 🏆
> - 教員が安全性・難易度・守秘義務を確認する ✅
> - 企業はテンプレートに沿って迷わず依頼できる 🏢
> - 企業と学生が相互評価してブラック案件を防ぐ ⭐
> - チームを組めば学科を越えた大型案件にも挑める 🤝

## 🚀 起動方法

### 方法1: Next.js 版(メイン)

```bash
npm install
npm run dev
```

### 方法2: スタンドアロン版(ビルド不要)

Node.js のインストールが不要な、素の HTML / CSS / JavaScript 版も同梱しています。
**`standalone/index.html` をブラウザで開くだけ**で同じ機能が動きます。
詳細は [`standalone/README.md`](standalone/README.md) を参照してください。

ブラウザで http://localhost:3000 を開いてください。

- バックエンド・DB・ログインは不要です(すべて **localStorage** に保存)
- 本番ビルドは `npm run build && npm run start`
- 進行状況をリセットしたい場合は、ブラウザのDevToolsで
  `localStorage.removeItem("mechaxmatch-state-v2")` を実行してリロードしてください

## 🛠 技術構成

| 項目 | 内容 |
| --- | --- |
| フレームワーク | Next.js 16(App Router)+ TypeScript |
| スタイリング | Tailwind CSS v4 |
| UIコンポーネント | shadcn/ui スタイルの自作コンポーネント(cva + tailwind-merge + lucide-react)※ |
| データ保存 | localStorage(経験値・追加クエスト・承認状態・完了教材・完了クエスト等) |
| 状態管理 | React Context(`src/lib/game.tsx`) |
| PDF出力 | 印刷用レイアウト + `window.print()` |

※ shadcn/ui の設計思想(cva によるバリアント、`cn()` ユーティリティ、Tailwindトークン)に沿って
`src/components/ui/` に button / card / badge / progress / tabs 等を実装しています。

## ✅ 実装済み機能一覧

### 画面(全14要素)
- **ホーム**: レベル・総経験値・連続学習日数・所持バッジ・おすすめクエスト/教材・週間ランキング・デイリークエスト・広告枠
- **クエスト一覧**: 難易度/報酬/XP/締切/リモート可否/安全レベル/危険度ラベル/教員承認ステータス等を表示、フィルタ付き
- **クエスト詳細**: 背景・依頼内容・成果物・評価基準・参考教材・提出テンプレート・企業評価・先輩の口コミ・挑戦/提出ボタン
- **企業向け投稿フォーム**: テンプレートに沿った5ステップ入力 → 投稿すると**「教員承認待ち」**でクエスト一覧に追加
- **学習ページ**: 10カテゴリ15教材、ミニクイズ(即時採点+解説)、関連クエストへのリンク、学習完了でXP獲得
- **スキルツリー**: 基礎/応用/総合の3ティア×10スキル、教材・クエスト完了で対応スキルのXPが増加しレベルアップ
- **プロフィール**: 完了クエスト・学習済み教材・バッジ・得意スキルTOP3・企業評価・ポートフォリオ実績一覧
- **教員承認画面**: 承認待ち一覧(危険度ラベル・NDA・使用設備等を表示)、コメント付きで承認/差し戻し/却下
- **提出テンプレート**: 8種類(CADデータ・実験レポート・設計レビュー等)、記入項目・形式・評価ポイント・記入例付き
- **相互評価**: 企業→学生(技術理解・納期遵守など5項目)/学生→企業(報酬妥当性・安全配慮など6項目)
- **スキル証明書**: 印刷用の白基調レイアウト、`window.print()` でPDF出力可能
- **チームクエスト**: 役割別の募集状況・必要スキル・チーム報酬/XPを表示、役割を選んで参加申請
- **広告枠**: ダミー広告6件をホーム・学習・クエスト詳細にPRラベル付きで表示
- **初回チュートリアル**: ゲームのプロローグ風。工学倫理→機密情報→安全管理→企業コミュニケーション→成果物品質の5つの心得+修了ミニテスト(6問・80%以上で合格)→ライセンス発行

### 見習い技術者ライセンス制度(受注ロック)
- チュートリアル未修了の学生は、企業クエスト・チームクエストを**受注できません**
- 未取得時は「挑戦する」が「🔒 ライセンス未取得」表示になり、クリックでチュートリアルへ誘導
- ミニテスト合格で **+300 XP** と「見習い技術者ライセンス」バッジを付与
- 不合格(80%未満)の場合は心得の復習ページへ戻されます
- ホームに「現在の受注資格」カードを表示(チュートリアル中 → 見習い技術者 → 企業クエスト受注可能 → 教員承認付き受注可能 → チームクエスト参加可能)

### 企業向け安心表示・教員向け管理強化
- 企業投稿画面: 「学生はチュートリアル修了後でないと受注できない」等の安心表示パネル、学生に公開する情報の**段階設定**(NDA後公開など)
- 教員画面: 学生ごとのチュートリアル/ライセンス取得状況、危険度の高い案件一覧、機密情報あり案件一覧、**承認時チェックリスト(7項目・全確認で承認ボタン有効化)**、差し戻し理由テンプレート

### ゲーミフィケーション
- 経験値 / レベルアップ(紙吹雪) / バッジ獲得(実績解除)/ ライセンス取得の各演出
- 称号(レベル帯で解放・コレクション表示) / アバター選択 / ギルドカード
- 連続学習ストリーク / 連続ログインボーナス(**宝箱のランダム報酬**+スタンプカード)
- デイリーミッション / ウィークリーミッション(進捗バー+達成報酬)
- レアクエスト(虹色ボーダー) / 緊急クエスト / 今週人気 / 「同級生が挑戦中」表示
- 期間限定イベントバナー / 企業からのスカウト風カード / 「あと少しでレベルアップ」表示
- ランキング3種(クラス内 / 全国高専 / スキル別)
- スキル診断(タイプ判定)+おすすめ進路表示

### コアフロー(動作確認済み)
1. 新規学生はまず**チュートリアル(ライセンス講習)**を修了 → ミニテスト合格でライセンス取得
2. 企業がフォームから投稿 → クエストが**教員承認待ち**で追加される
3. 教員がチェックリスト確認のうえ承認 → 学生のクエストボードで**「挑戦する」が有効化**される
4. 学生が挑戦 → 成果物提出 → **XP・スキルXP獲得**、実績がプロフィール/証明書に反映
5. すべての進行状況が localStorage に保存され、リロード後も維持される

### localStorageに保存される主なデータ
`xp` / `weeklyXp` / `skillXp` / `loginStreak` / `completedMaterials` / `completedQuests` /
`customQuests`(企業投稿クエスト) / `statusOverrides`(承認状態) / `teamApplications` /
`tutorialCompleted` / `licenseIssued` / `licenseIssuedAt` / `tutorialQuizScore` /
`completedTutorialModules` / `earnedBadges` / `dailyMissionProgress` / `weeklyMissionProgress` /
`unlockedTitles` / `studentStatus` / `avatar` / `loginStamps` ほか(キー: `mechaxmatch-state-v2`)
