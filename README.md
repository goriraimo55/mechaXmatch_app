# ⚙️ MechaXMatch — 高専生の技術者ギルド

高専生がゲーム感覚で機械設計・ものづくり・CAD・実験・加工・AI/DXなどの実務スキルを学びながら、
企業から投稿された小さな技術タスクに「クエスト」として挑戦できる、学習 × 仕事マッチングWebアプリの**プロトタイプ**です。

> このアプリの本質は、**「企業の小さな技術課題」を「高専生の成長クエスト」に変換する**ことです。
>
> - 仕事をすると経験値が貯まる ⚔️
> - 学ぶとスキルが伸びる 📖
> - 実績がポートフォリオ・スキル証明書になる 🏆
> - 教員が安全性・難易度・守秘義務を確認する ✅
> - 企業はテンプレートに沿って迷わず依頼できる 🏢
> - 企業と学生が相互評価してブラック案件を防ぐ ⭐
> - チームを組めば学科を越えた大型案件にも挑める 🤝

## 🚀 起動方法

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。

- バックエンド・DB・ログインは不要です(すべて **localStorage** に保存)
- 本番ビルドは `npm run build && npm run start`
- 進行状況をリセットしたい場合は、ブラウザのDevToolsで
  `localStorage.removeItem("mechaxmatch-state-v1")` を実行してリロードしてください

## 🛠 技術構成

| 項目 | 内容 |
| --- | --- |
| フレームワーク | Next.js 16(App Router)+ TypeScript |
| スタイリング | Tailwind CSS v4 |
| UIコンポーネント | shadcn/ui スタイルの自作コンポーネント(cva + tailwind-merge + lucide-react)※ |
| データ保存 | localStorage(経験値・追加クエスト・承認状態・完了教材・完了クエスト等) |
| 状態管理 | React Context(`src/lib/game.tsx`) |
| PDF出力 | 印刷用レイアウト + `window.print()` |

※ shadcn/ui の設計思想(cva によるバリアント、`cn()` ユーティリティ、Tailwindトークン)に沿って
`src/components/ui/` に button / card / badge / progress / tabs 等を実装しています。

## ✅ 実装済み機能一覧

### 画面(全13要素)
- **ホーム**: レベル・総経験値・連続学習日数・所持バッジ・おすすめクエスト/教材・週間ランキング・デイリークエスト・広告枠
- **クエスト一覧**: 難易度/報酬/XP/締切/リモート可否/安全レベル/危険度ラベル/教員承認ステータス等を表示、フィルタ付き
- **クエスト詳細**: 背景・依頼内容・成果物・評価基準・参考教材・提出テンプレート・企業評価・先輩の口コミ・挑戦/提出ボタン
- **企業向け投稿フォーム**: テンプレートに沿った5ステップ入力 → 投稿すると**「教員承認待ち」**でクエスト一覧に追加
- **学習ページ**: 10カテゴリ15教材、ミニクイズ(即時採点+解説)、関連クエストへのリンク、学習完了でXP獲得
- **スキルツリー**: 基礎/応用/総合の3ティア×10スキル、教材・クエスト完了で対応スキルのXPが増加しレベルアップ
- **プロフィール**: 完了クエスト・学習済み教材・バッジ・得意スキルTOP3・企業評価・ポートフォリオ実績一覧
- **教員承認画面**: 承認待ち一覧(危険度ラベル・NDA・使用設備等を表示)、コメント付きで承認/差し戻し/却下
- **提出テンプレート**: 8種類(CADデータ・実験レポート・設計レビュー等)、記入項目・形式・評価ポイント・記入例付き
- **相互評価**: 企業→学生(技術理解・納期遵守など5項目)/学生→企業(報酬妥当性・安全配慮など6項目)
- **スキル証明書**: 印刷用の白基調レイアウト、`window.print()` でPDF出力可能
- **チームクエスト**: 役割別の募集状況・必要スキル・チーム報酬/XPを表示、役割を選んで参加申請
- **広告枠**: ダミー広告6件をホーム・学習・クエスト詳細にPRラベル付きで表示

### ゲーミフィケーション
- 経験値 / レベルアップ(紙吹雪アニメーション) / 称号(レベル帯で変化)
- 連続学習日数(ストリーク) / バッジ10種(教員承認済みバッジ・企業評価バッジ含む)
- 週間ランキング / デイリークエスト / レアクエスト(虹色ボーダー演出)
- 学習完了・クエスト完了時のXP獲得アニメーション

### コアフロー(動作確認済み)
1. 企業がフォームから投稿 → クエストが**教員承認待ち**で追加される
2. 教員が承認 → 学生のクエストボードで**「挑戦する」が有効化**される
3. 学生が挑戦 → 成果物提出 → **XP・スキルXP獲得**、実績がプロフィール/証明書に反映
4. すべての進行状況が localStorage に保存され、リロード後も維持される

## 🔮 今後追加すべき機能リスト

- **認証・ロール管理**: 学生/企業/教員の実ログイン(高専機構SSO・企業アカウント審査)

- **提出物レビュー機能**: 企業からのフィードバック・差し戻し・再提出の往復管理
- **通知**: 承認・締切・レビューのプッシュ/メール通知、締切リマインダー
- **マッチング高度化**: スキルレベルに応じたクエスト推薦、難易度自動判定

- **証明書の信頼性強化**: 発行番号・検証用QRコード・改ざん防止(署名付きPDF)
- **学校側管理画面**: 学科・学年ごとの活動状況ダッシュボード、安全インシデント報告
- **広告配信基盤**: 実広告の入稿・計測(インプレッション/クリック)
- **多言語対応・アクセシビリティ改善**
