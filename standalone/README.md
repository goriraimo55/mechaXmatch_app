# ⚙️ MechaXMatch — スタンドアロン版(HTML / CSS / JavaScript)

Next.js 版と同じ機能・デザインを、**素の HTML / CSS / JavaScript だけ**で実装したバージョンです。
ビルドもサーバも不要で、`index.html` をブラウザで開くだけで動きます。

## 🚀 起動方法

```
standalone/index.html をダブルクリックしてブラウザで開くだけ
```

- インストール・ビルド・サーバ起動は一切不要です(`file://` で動作確認済み)
- データの保存には localStorage を使用します(キー: `mechaxmatch-standalone-v1`)
- 進行状況はサイドバー下の「🔄 進行状況をリセット」ボタンで初期化できます
- Next.js 版とは保存キーが別なので、両方を並行して試せます

## 📁 ファイル構成

```
standalone/
├── index.html   # エントリーポイント(レイアウト・ナビ・演出オーバーレイ)
├── style.css    # ダークネオンテーマ(グリッド背景・ガラスモーフィズム・印刷CSS)
├── app.js       # アプリ本体(状態管理・ハッシュルーター・全13画面・演出)
└── data.js      # ダミーデータ(Next.js版 src/lib/data/*.ts から自動変換した同一データ)
```

## 実装メモ

- フレームワーク・ライブラリ不使用(依存ゼロ)。アイコンは絵文字で代替しています
- 画面はテンプレート文字列によるフルレンダリング方式(`render()`)で、
  状態変更のたびに再描画するシンプルな構成です
- `data.js` は Next.js 版の TypeScript データを `tsc` で変換して生成しており、
  クエスト20件・教材15件・バッジ11種などの内容は完全に同一です
- 経験値・レベル・ライセンス・承認状態などの仕様も Next.js 版と同じです
